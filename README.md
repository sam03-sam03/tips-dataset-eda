# Tips Dataset EDA

Exploratory data analysis on restaurant tipping behavior — uncovering patterns 
by day, group size, and other factors using Python, pandas, and seaborn.

## Dataset
The built-in seaborn `tips` dataset — 244 restaurant bills with details on 
total bill, tip amount, day, time, group size, and more.

## Key Findings
- **Day of week matters:** Friday has the highest average tip percentage 
  (16.99%), while Saturday has the lowest (15.31%) — despite Saturday 
  likely being the busiest night.
- **Group size affects tip %:** Larger groups tend to tip a lower 
  percentage of the bill compared to smaller groups, suggesting tips 
  don't scale proportionally with bill size.

## Tools Used
- Python
- pandas
- seaborn / matplotlib
